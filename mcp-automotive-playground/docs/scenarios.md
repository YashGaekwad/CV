# Scenario Pack

## 1) Check engine light triage

**Intent:** Understand fault and produce actionable next steps.

**Tool chain:**
- Diagnostics → detect OBD-II code and severity
- Knowledge → decode and explain code
- Maintenance → suggest service actions

## 2) Route risk advisory

**Intent:** Identify route and weather-linked hazards before departure.

**Tool chain:**
- Navigation → route and ETA
- Weather → severe conditions on route window
- Emergency → contingency recommendations

## 3) Pre-trip health check

**Intent:** Confirm vehicle readiness before long drive.

**Tool chain:**
- Vehicle Info → current profile and mileage
- Maintenance → overdue items
- Diagnostics → basic health scan

## 4) Safe-driving coach

**Intent:** Improve safety decisions in changing conditions.

**Tool chain:**
- Weather → traction/visibility risks
- Navigation → road context
- Emergency → escalation guidance
- Knowledge → explain why recommendations matter
